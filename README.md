# Resume Analyzer

This project uses spaCy and machine learning tools to analyze and extract key insights from resumes.

## 💡 Features
- Named Entity Recognition (NER)
- Tokenization
- POS tagging

## 📂 Requirements
- Python 3.10
- spaCy 3.5.3
- scikit-learn
- pandas
- tqdm
